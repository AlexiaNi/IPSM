# IPSM

   IPSM - "Indoor Plant Monitoring System" este un sistem automatizat pentru monitorizarea plantelor de casa.     
   Componente:   
          - DHT11  
          - Senzor Higrometru   
          - Arduino R3   
          - Accesorii   
          - Plexiglas   
          - LED    
          - Pompa persitaltica   
          - Modul releu   
          - LCD 16x2   
          - Senzor nivel apa
